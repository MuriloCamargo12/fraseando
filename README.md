# 📕 Fraseando

Aplicação web que gera frases aleatórias com tradução automática em tempo real e funcionalidades de interação como favoritar e compartilhar.

O projeto foi desenvolvido com foco em praticar consumo de APIs e tratamentos de erros. Utilizando Next.js, React, TypeScript e APIs (Zenquotes e Langbly).

---

## 🚀 Deploy

🔗 **Acesse o projeto online:**  
https://fraseando-six.vercel.app/

---

## ▶️ Como rodar o projeto localmente

```bash
# Clonar o repositório
git clone https://github.com/MuriloCamargo12/fraseando.git

# Entrar na pasta do projeto
cd fraseando

# Acesse https://langbly.com/ para gerar uma KEY da API

# Criar arquivo .env na raiz com:

TRANSLATE=SUA_API_KEY_DA_LANGBLY

# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```

## ⚙️ Funcionalidades

- 🔎 Geração de frases aleatórias
- 🌐 Opção de tradução automática das frases (EN → PT)
- ❤️ Opção de favoritar frases
- 📧 Opção de compartilhar frase no Whatsapp
- ❌ Tratamento de erros da API
- 💾 Persistência de favoritos e tradução com localStorage
- ⏳ Estados de loading
- 📱 Layout responsivo (mobile e desktop)
- ✨ Interface moderna com animações suaves
